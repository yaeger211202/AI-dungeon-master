# AI Dungeon Master

An AI-powered interactive storytelling system built with a Large Language Model (LLM) as the narrative engine.

## Project Description
AI Dungeon Master is a Generative AI system for structured and coherent interactive storytelling. The system uses a pre-trained LLM to generate dynamic story narratives while maintaining persistent player state across multi-turn gameplay sessions.

## Features
- LLM-powered narrative generation
- Persistent player state tracking (health, inventory, quests, location)
- Multi-turn conversation history
- Interactive gameplay loop

## Tech Stack
- Python
- HuggingFace Transformers
- TinyLlama-1.1B (baseline model)
- Google Colab
